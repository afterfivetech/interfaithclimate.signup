interfaithclimate.signup Installation
-------------------------------------

To install interfaithclimate.signup using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``interfaithclimate.signup`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        interfaithclimate.signup

* Re-run buildout, e.g. with:

    $ ./bin/buildout
